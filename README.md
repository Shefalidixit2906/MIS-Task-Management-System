# 🚀 MIS Task Management System

## 📌 Project Overview

This project is a complete MIS (Management Information System) built using Google Sheets and Apps Script.

It automates task assignment, employee tracking, and real-time updates.

---

## ⚙️ Features

* Auto Task ID Generation
* Auto Date & Time Tracking
* Employee Task Sheets Creation
* Real-Time Sync (Employee → Master)
* Status & Remarks Update
* Completion Time Automation
* Delay Days Calculation
* Completed Tasks Auto Move

---

## 🔄 Workflow

1. Manager assigns task
2. Employee gets task sheet
3. Employee updates status
4. System auto updates:

   * Completion time
   * Master sheet
   * Delay tracking

---

## 🛠️ Tech Used

* Google Sheets
* Google Apps Script
